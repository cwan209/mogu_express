# 演示录屏脚本(3-5 分钟)

适合录屏发给甲方,或对内 stand-up 演示。

## 准备

确保已起:
- `cd local-backend && docker compose up -d` ✓
- `cd web-admin && npm run dev`(http://localhost:5173)
- 微信开发者工具打开项目,Cmd+B 编译

**两个窗口并排开**:
- 左:微信开发者工具(模拟器)
- 右:浏览器 Web 后台

录屏建议用 macOS 自带 QuickTime 的"屏幕录制",或 Loom。

## 演讲稿(对应每个步骤念出来)

### 开场(15 秒)

> "这是 mogu_express 接龙团购小程序。对标快团团,服务澳洲华人社区,
> 单商家模式。今天演示完整产品闭环。"

### 第一幕:顾客视角(90 秒)

**[切到微信开发者工具]**

> "顾客打开小程序,首页是团长发布的'团'列表 — 现在有 3 个团:
> 本周生鲜、烧烤肉类、即将开始的乳制品团。每张卡片显示倒计时
> 和团状态。"

→ **点击"本周生鲜团"**

> "进入团详情,看到团内 3 个商品。"

→ **点击"塔斯马尼亚蓝莓"**

> "商品详情页,这里是产品的灵魂功能 — **已订购参与者名单**。
> 14 个人在我之前接龙了,这种社交压力会促进下单。"

→ **滑到底,点"加入购物车"**(toast: 已加入)

> "加入购物车。再加一件西兰花。"

→ **返回 → 西兰花 → 加购**

→ **底部 tab 切到"购物车"**

> "购物车,可以多选 / 改数量 / 删除。这两件勾选合计 14.98 澳币,
> 点结算。"

→ **结算**

> "订单确认页,需要先完善姓名电话和地址。"

→ **添加地址 → 完善资料 → 返回 → 提交订单**

→ **跳到支付结果页**

> "现在是 HuePay 支付环节,因为商户凭证还没到位,我们用 stub 模式 —
> 点'模拟支付成功'代替真实支付。"

→ **点模拟支付** → 变绿色 ✓

> "支付完成。切到订单 tab 看到这单已支付。"

→ **底部 tab 切到"订单"** → 看到刚才的订单

### 第二幕:团长视角(90 秒)

**[切到浏览器]**

> "团长在 Web 后台看运营。仪表盘显示今日 GMV、订单数、热销商品、
> 各团表现。"

→ **指着今日订单数 / GMV 数字** → 比之前 +1

> "刚才顾客的下单立即同步到这里 — 这是真实的 MongoDB,
> 手机和后台是同一份数据。"

→ **点"订单"导航** → 看到刚才的订单

> "订单管理,可以按状态/团/日期筛选。这单是'已支付',点'发货'。"

→ **点发货按钮** → 状态变蓝色"已发货"

→ **切回微信开发者工具,订单详情页下拉刷新或重进**

> "顾客这边马上能看到'已发货'。三端实时同步。"

→ **回到 Web 后台 → 订单页 → 点"导出 (N)"**

> "团长一键导出订单 Excel,两个 sheet — 订单明细 + 商品销量汇总。"

→ **打开下载的 xlsx,展示两个 sheet**

### 第三幕:产品管理(60 秒)

**[Web 后台]**

→ **团管理 → 新建团**

> "团长在 Web 后台发新团。填标题、描述、封面 URL、开始/截止时间。"

→ **保存** → **进商品管理 → 新建商品 → 选刚才的团 → 填价格库存 → 保存**

→ **切回小程序首页 → 下拉刷新**

> "新团立刻出现在小程序首页。"

### 第四幕:分享与海报(45 秒)

**[小程序商品详情页]**

→ **点底栏"分享"按钮** → 弹出微信原生分享面板

> "原生分享卡片可以转发到群和好友。"

→ **点"海报"**

> "再来海报模式 — canvas 实时合成商品图、价格、二维码,
> 可保存到相册发朋友圈。"

→ **等海报画好** → **点保存到相册** → "已保存" toast

### 第五幕:小程序内的团长管理台(30 秒)

→ **小程序"我的"tab → 团长管理台 → 以管理员身份预览**

> "团长出门在外用手机,小程序里也有简化版后台 —
> 看今日 GMV、待发货订单,可一键拨打顾客电话、标记发货。"

### 收尾(20 秒)

> "产品功能开发完毕。本地完整环境跑通。
> 上线还差三件东西需要客户提供 —
>  1. 正式小程序 AppID
>  2. HuePay 商户凭证
>  3. VPS 服务器
> 详见现状报告。"

## 录屏 tips

- 录之前关闭无关通知,把 dock/menu bar 自动隐藏
- Web 后台先重置一次 mock 数据(Dashboard 右上角"重置 Mock"),保证统计数字干净
- 小程序提前清掉旧订单(可以删 mock_orders storage 或用 docker `db.orders.deleteMany({})`)
- 推荐 1080p 录,后期不缩放最清晰
- 加字幕(QuickTime 不支持,可上传 Loom 自动转写)

## 简短版本(1 分钟极速版)

如果只想给一个 1 分钟剪辑:

- 0:00-0:15 — 首页 → 团 → 商品(秀已订购名单)
- 0:15-0:30 — 加购 → 结算 → 模拟支付成功
- 0:30-0:45 — 切 Web 后台,看 GMV +1,点发货
- 0:45-1:00 — 切回小程序看到"已发货",收尾"三端实时同步,产品已就绪"
