## 1. 项目是什么

对标 **快团团** 的接龙式团购微信小程序,服务**澳洲华人社区**:

- **顾客**:在小程序里浏览团长发布的"团",加购物车下单,**澳币(AUD)在线支付**,跟踪订单
- **团长**:Web 后台管理团/商品/订单 + 一键导出 Excel + 小程序里实时看待发货订单 + 标记发货
- **支付**:对接 **HuePay**(澳洲持牌聚合支付,通微信支付/支付宝)

## 4. 演示流程(3 分钟脚本)

### 顾客视角
1. 打开小程序首页 → 看到 3 张"团"卡片(本周生鲜 / 烧烤肉 / 乳制品)
2. 点"本周生鲜团" → 团详情页 → 看到 3 个商品
3. 点"塔斯马尼亚蓝莓" → 商品详情页(图轮播 + 已 14 人接龙 + 加购按钮)
4. 加购 → 切到购物车 tab → 勾选 → 结算
5. 添加收货地址 → 完善姓名电话 → 提交订单
6. 跳支付页 → 点"模拟支付成功" → 变绿色 ✓
7. 切到订单 tab → 看到这单"已支付"

### 团长视角
8. 浏览器打开 Web 后台 → 仪表盘 → 看到刚才的订单已计入今日 GMV
9. 订单管理 → 看到这单 → 点"发货" → 状态变蓝色"已发货"
10. **手机回去看** → 订单详情自动显示"已发货"
11. 团管理 → 新建一个团 → 商品管理 → 加几个商品 → **小程序首页立刻多一个团卡片**
12. 订单管理 → 选筛选 → 点**"导出 (N)"** → 下载 Excel(两 sheet:明细 + 销量汇总)

## 附录:技术决策清单(给客户技术顾问看)

| 决策 | 选择 | 理由 |
|---|---|---|
| 小程序框架 | 原生 JS + WXML(非 uni-app/Taro) | 性能好、TDesign 适配最佳、单端项目无需跨端 |
| UI 库 | TDesign Miniprogram | 腾讯出品、Material 风格、覆盖完整 |
| 后端 | Express + MongoDB(不强制云开发) | 测试号不能用云开发,且自建给我们更多控制权 |
| Web 后台 | Vite + React + antd | 开发快、构建产物小、antd 组件齐 |
| 支付 | HuePay 聚合 | 客户指定;对比 Airwallex/Pingpong 等,HuePay 文档稍少但承诺一致 |
| 本地开发 | Docker compose + wx-server-sdk shim | 无需真云环境就能开发,加快迭代 |
| 部署 | VPS + Docker + Nginx | 成本低、可控、易扩展 |
