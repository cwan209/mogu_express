### 部署路径预备(分支 A 或 B 都需要)

云开发禁用后,生产路径锁定为**自建 HTTP 后端 + 公网部署**:
- VPS:腾讯云轻量 香港(¥24/月,海外主体可注册腾讯云国内版用 Visa 卡 + 经营者实名)
- 域名:任一 `.com`,DNS 走 Cloudflare 拿免费 SSL
- 反代:Caddy(自动 Let's Encrypt)
- 对象存储:腾讯云 COS 香港(同账号集成,海外主体可开)
- 微信公众平台:服务器域名白名单加入 api 域名 + COS 域名

VPS 选型对比和域名/账号路径在用户的 AskUserQuestion 答复中**未最终敲定**,等商业模式确认后再回过头决定。
